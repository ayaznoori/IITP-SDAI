# 1️⃣ What is Programming?

Programming means giving instructions to a computer.

Think of it like writing a recipe:

 Step 1 — Take ingredients
 Step 2 — Mix them
 Step 3 — Cook

Similarly, a program tells the computer:

* Take input
* Process data
* Show output

You do not need any prior coding experience to start learning Python.

---

# 2️⃣ What is Python?

Python is a programming language designed to be simple and readable.

It is used by beginners as well as professional developers.

Python is popular because:

* Easy syntax
* Less complex rules
* Used in real industries like AI, Web Development, and Automation

Example of Python code:

```python
print("Hello World")
```

This single line tells the computer to display text on the screen.

# 4️⃣ Basic Programming Words to Know

Before class, try to understand these simple terms:

### Variable

A container that stores information.

Example idea:

```
name = "Ali"
```

Here, `name` stores text.

---

### Data Type

Type of data stored in a variable.

Common examples:

* Text → `"Hello"`
* Number → `10`
* Decimal → `10.5`
* True/False → `True`

---

### Input and Output

Programs often interact with users.

* **Input** → user gives data
* **Output** → program shows result

Example idea:

```python
name = input("Enter your name")
print(name)
```
